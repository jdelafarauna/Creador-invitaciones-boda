# 💍 Constructor de Invitaciones de Boda Pro

Esta es una herramienta interactiva, visual y modular para crear invitaciones de boda elegantes directamente desde tu navegador, sin necesidad de saber programación ni usar herramientas externas como Canva.

## ✨ Características Principales

- **Edición Visual Directa**: Haz clic en cualquier texto para cambiar su contenido.
- **Personalización de Estilos**: Cada bloque de texto tiene un panel de estilo (🎨) para cambiar:
  - **Color**: Selector de color infinito.
  - **Tamaño**: Ajuste preciso de la letra.
  - **Tipografía**: Biblioteca de +20 fuentes premium (Elegantes, Modernas, Románticas y Manuscritas).
- **Sistema Modular de Secciones**:
  - Añade secciones al final o **insértalas entre dos existentes** pinchando en los disparadores dorados (+).
  - Elimina secciones completas con un solo clic.
- **Contenido Multimedia**:
  - **Fotos**: Sube tus propias imágenes de portada o detalles.
  - **Fondos**: Cambia el color de fondo de cada sección o sube una imagen para que cubra todo el bloque.
  - **Mapas Reales**: Inserta mapas interactivos de Google Maps simplemente escribiendo la dirección.
- **Diseño de Columnas**: Crea layouts sofisticados con contenido lado a lado que se adapta automáticamente a móviles.
- **Exportación Limpia**: Al terminar, descarga un archivo HTML único con todo incrustado, listo para enviar por WhatsApp o subir a un servidor.

## 🚀 Cómo empezar

1. Descarga este repositorio.
2. Abre el archivo `index.html` en tu navegador favorito (Chrome, Safari, Edge, etc.).
3. Empieza a editar la plantilla base o crea la tuya desde cero.
4. Usa los controles inferiores para cambiar los colores globales de fondo.
5. Cuando tu diseño sea perfecto, pulsa el botón verde **"GUARDAR DISEÑO"**.

## 🛠️ Herramientas de Edición

- **🎨 Pincel**: Abre el menú de estilo del elemento.
- **🗑️ Papelera**: Elimina ese elemento o sección específica.
- **➕ Botones de Sección**: Permiten añadir Texto (+T), Título (+H), Imagen (+I), Mapa (+M) o Columnas.
- **Líneas de Inserción**: Pasa el ratón entre dos secciones para insertar una nueva ahí mismo.

## 📦 Exportación

Al pulsar "Guardar", la herramienta realiza una limpieza profunda del código:
- Elimina todos los botones de edición.
- Quita las líneas de ayuda y bordes de diseño.
- Desactiva el modo edición para que los invitados no puedan borrar nada.
- Genera un archivo `.html` optimizado.
